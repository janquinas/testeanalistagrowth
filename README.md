# testeanalistagrowth
Este projeto consiste numa solução desenvolvida em Java puro que automatiza a análise de testes A/B de cashback para o time de Growth do Méliuz. A solução utiliza engenharia de prompts avançada ligada à API do Google Gemini (`gemini-3.5-flash`) para processar ficheiros de dados brutos e extrair insights acionáveis de negócio.
